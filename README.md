## 📌 About the Project

This project is a multi-section landing page designed based on a Figma layout.

The goal was to practice semantic HTML, CSS styling, layout techniques, and responsive design principles.

The page includes sections such as hero, services, team, portfolio, and footer.

## 🛠 Tech Stack

- HTML5
- CSS3
- Flexbox
- CSS variables
- Git & GitHub
- GitHub Pages
- Figma design reference

## ✨ Features

- Semantic HTML structure
- Responsive layout
- Flexbox-based positioning
- Navigation menu
- Hero section with call-to-action
- Services and team sections
- Portfolio grid
- Footer with contacts
- Hover and focus effects
- Clean and structured CSS

- Working with Figma designs
